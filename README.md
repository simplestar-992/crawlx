# CRAWLX

**Web Crawler with Link Extraction**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/crawlx.git
cd crawlx
go build -o crawlx .
```

## Usage

```bash
./crawlx --help
```
