## ✨ Features

- 🚀 Blazing fast performance
- 💎 Clean and intuitive API
- 🔧 Zero configuration needed
- 🛡️ Secure by default
- 📦 Single binary, no dependencies

## 📥 Installation

```bash
git clone https://github.com/simplestar-992/crawlx.git
cd crawlx
go build -o crawlx .
./crawlx --help
```

## 🚀 Quick Start

```bash
./crawlx -h
```

## 🧪 Examples

```bash
# Example 1
./crawlx status

# Example 2  
./crawlx --verbose
```
